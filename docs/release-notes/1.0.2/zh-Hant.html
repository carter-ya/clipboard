Clipboard 1.0.2

優化

  • 偏好設定新增獨立的 AI 分頁：原 General 底部的『AI Summaries』開關組（圖片 / 文字 / 檔案 + 本機能力說明）整體上移為頂層 Tab，AI 設定一步直達。
  • 歷史面板底部的快速鍵 footer 移除：同一份資訊已經在 Preferences → Shortcuts 分頁裡完整列出，面板整體高度收緊約 29pt，列表區域自然多顯示 1~2 條。

修復

  • 偏好設定 → Updates 區塊的標題、『檢查更新…』按鈕、未配置提示現在會跟隨語言，不再硬編碼英文。
  • Sparkle 升級彈窗現在依系統語言顯示 release notes（en / zh-Hans / zh-Hant / ja / ko / de / es），此前彈窗為空白。

分發

  • 一鍵安裝腳本：curl -fsSL https://carter-ya.github.io/clipboard/install.sh | bash —— 抓取最新 DMG、校驗 SHA-256、退出執行中的 Clipboard、安裝到 /Applications/、清除 Gatekeeper 隔離標記。
