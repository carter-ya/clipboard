Clipboard 1.0.2

改善

  • 環境設定に専用の AI タブを追加：これまで General タブの末尾にあった「AI 要約」のスイッチ群（画像 / テキスト / ファイル + デバイスの対応状況）を最上位の Tab に昇格し、AI の設定に直接アクセスできるようになりました。
  • 履歴パネル下部のショートカット footer を削除：同じ情報は Preferences → Shortcuts に揃っているため、パネルの高さが約 29pt 縮まり、リスト領域で自然に 1〜2 件多く表示されます。

修正

  • 環境設定 → Updates セクションの見出し、「アップデートを確認…」ボタン、未設定時のメッセージがシステム言語に追従するようになりました（以前は英語のまま）。
  • Sparkle のアップデート通知がシステム言語のリリースノート（en / zh-Hans / zh-Hant / ja / ko / de / es）を表示するようになりました。従来は空白でした。

配布

  • ワンライナーインストーラー：curl -fsSL https://carter-ya.github.io/clipboard/install.sh | bash — 最新の DMG を取得し、SHA-256 を検証、実行中の Clipboard を終了、/Applications/ にインストール、Gatekeeper の検疫属性をクリアします。
