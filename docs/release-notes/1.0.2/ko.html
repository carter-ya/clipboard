Clipboard 1.0.2

개선

  • 환경설정에 전용 AI 탭을 추가했습니다. 기존에 General 탭 하단에 있던 ‘AI 요약’ 스위치들(이미지 / 텍스트 / 파일 + 기기 지원 현황)을 최상위 Tab 으로 승격하여 AI 설정에 바로 접근할 수 있습니다.
  • 기록 패널 하단의 단축키 footer 를 제거했습니다. 같은 정보가 이미 Preferences → Shortcuts 탭에 모두 정리되어 있어, 패널 높이가 약 29pt 줄어들고 목록 영역이 1~2 행 더 표시됩니다.

수정

  • 환경설정 → Updates 섹션의 제목, ‘업데이트 확인…’ 버튼, 미설정 안내가 시스템 언어를 따라갑니다 (이전에는 영문 고정).
  • Sparkle 업데이트 대화상자가 시스템 언어의 릴리즈 노트(en / zh-Hans / zh-Hant / ja / ko / de / es)를 표시합니다. 이전에는 내용이 비어 있었습니다.

배포

  • 원라인 설치 스크립트: curl -fsSL https://carter-ya.github.io/clipboard/install.sh | bash — 최신 DMG 다운로드, SHA-256 검증, 실행 중인 Clipboard 종료, /Applications/ 에 설치, Gatekeeper 격리 속성 제거를 자동 처리합니다.
