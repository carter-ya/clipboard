Clipboard 1.0.2

Verbessert

  • Die Einstellungen haben jetzt einen eigenen KI-Tab: Die Schalter für KI-Zusammenfassungen (Bilder / Text / Dateien + Hinweis zur Geräteunterstützung) stehen nicht mehr am Ende von „Allgemein“, sondern als eigener Tab gleichrangig neben Allgemein / Tastenkürzel / Datenschutz / Daten — die KI-Konfiguration ist einen Klick entfernt.
  • Die Shortcut-Fußzeile am unteren Rand des Verlaufs-Panels entfällt; dieselbe Übersicht liegt bereits unter Einstellungen → Tastenkürzel. Das Panel ist rund 29pt schlanker, die Liste zeigt 1–2 Einträge mehr.

Behoben

  • Einstellungen → Updates: Abschnittstitel, die Schaltfläche „Nach Updates suchen…“ und der Hinweis zu nicht konfigurierten Updates folgen jetzt der Systemsprache (vorher durchgängig Englisch).
  • Der Sparkle-Aktualisierungsdialog zeigt jetzt Release Notes in der Systemsprache (en / zh-Hans / zh-Hant / ja / ko / de / es). Zuvor blieb der Dialog leer.

Distribution

  • Einzeiliger Installer: curl -fsSL https://carter-ya.github.io/clipboard/install.sh | bash — lädt das neueste DMG, prüft den SHA-256, beendet ein laufendes Clipboard, installiert nach /Applications/ und entfernt das Gatekeeper-Quarantäne-Attribut.
