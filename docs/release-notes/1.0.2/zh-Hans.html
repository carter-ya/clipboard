Clipboard 1.0.2

优化

  • 偏好设置新增独立的 AI 分页：原 General 底部的『AI Summaries』开关组（图片 / 文本 / 文件 + 本机能力说明）整体上移为顶层 Tab，AI 配置一步直达。
  • 历史面板底部的快捷键 footer 移除：同一份信息已经在 Preferences → Shortcuts 分页里完整列出，面板整体高度收紧约 29pt，列表区域自然多显示 1~2 条。

修复

  • 偏好设置 → Updates 区块的标题、『检查更新…』按钮、未配置提示现在会跟随语言，不再硬编码英文。
  • Sparkle 升级弹窗现在按系统语言显示 release notes（en / zh-Hans / zh-Hant / ja / ko / de / es），此前弹窗为空白。

分发

  • 一键安装脚本：curl -fsSL https://carter-ya.github.io/clipboard/install.sh | bash —— 拉最新 DMG、校验 SHA-256、退出正在运行的 Clipboard、安装到 /Applications/、清 Gatekeeper 隔离标记。
