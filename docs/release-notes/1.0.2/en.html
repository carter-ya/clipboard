Clipboard 1.0.2

Improved

  • Preferences now has a dedicated AI tab — the summary toggles (images / text / files) and capability caption were promoted out of General so the AI setup is one click away.
  • The history panel footer of shortcut hints is gone; the same shortcuts are already listed under Preferences → Shortcuts. The panel is ~29pt shorter, showing 1–2 more rows.

Fixed

  • Preferences → Updates now localizes the section title, the “Check for Updates…” button and the “Automatic updates are not configured in this build.” notice. Previously these stayed in English regardless of system language.
  • The Sparkle update dialog now shows release notes in your system language (en / zh-Hans / zh-Hant / ja / ko / de / es). Previously the dialog was blank.

Distribution

  • One-line installer: curl -fsSL https://carter-ya.github.io/clipboard/install.sh | bash — grabs the latest DMG, verifies the SHA-256, quits any running Clipboard, installs to /Applications/, and strips the Gatekeeper quarantine bit.
