Clipboard 1.0.3

Novedades

  • Motor de IA remoto opcional para resúmenes de texto e imagen — apunta Clipboard a cualquier endpoint compatible con OpenAI (OpenAI, Ollama, vLLM, OpenRouter, DeepSeek, etc.).
  • Si la llamada remota falla, se recurre automáticamente a Foundation Models / NaturalLanguage / Vision en el dispositivo.
  • Activación opcional, desactivado por defecto. La clave de API se guarda en el Llavero por endpoint. Los elementos sensibles y los del portapapeles marcados como «concealed» nunca se envían a un servicio remoto.
  • Localizado en 7 idiomas.

Rendimiento

  • Seleccionar elementos del portapapeles con texto muy grande (varios MB) ya no bloquea la interfaz. La vista previa ahora se renderiza de forma asíncrona y se limita a 64 KB; el portapapeles sigue recibiendo el contenido completo.
