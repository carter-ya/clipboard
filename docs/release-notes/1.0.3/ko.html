Clipboard 1.0.3

새 기능

  • 텍스트와 이미지 요약을 위한 선택적 원격 AI 엔진 — OpenAI 호환 엔드포인트(OpenAI, Ollama, vLLM, OpenRouter, DeepSeek 등)를 지정할 수 있습니다.
  • 원격 호출이 실패하면 기기 내장의 Foundation Models / NaturalLanguage / Vision 으로 자동 폴백됩니다.
  • 기본값은 꺼짐(opt-in). API 키는 엔드포인트별로 Keychain 에 저장됩니다. 민감 항목과 concealed 클립보드 항목은 원격으로 전송되지 않습니다.
  • 7 개 언어로 현지화.

성능

  • 텍스트가 매우 큰(수 MB) 클립보드 항목을 선택할 때 UI가 멈추지 않습니다. 미리보기는 비동기로 렌더링되며 표시는 64 KB까지로 제한됩니다. 클립보드에는 여전히 전체 내용이 복사됩니다.
