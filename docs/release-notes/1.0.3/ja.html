新機能

  • テキストと画像の要約に使えるオプションのリモート AI エンジン — OpenAI 互換エンドポイント（OpenAI、Ollama、vLLM、OpenRouter、DeepSeek など）を指定できます。
  • リモート呼び出しに失敗した場合は、オンデバイスの Foundation Models / NaturalLanguage / Vision に自動でフォールバックします。
  • オプトイン、デフォルトはオフ。API キーはエンドポイントごとにキーチェーンに保存されます。機密項目や concealed なクリップボード項目はリモートには送信されません。
  • 7 言語にローカライズ。
