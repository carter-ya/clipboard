Clipboard 1.0.3

What's new

  • Optional remote AI engine for text and image summaries — point Clipboard at any OpenAI-compatible endpoint (OpenAI, Ollama, vLLM, OpenRouter, DeepSeek, etc.).
  • Falls back to on-device Foundation Models / NaturalLanguage / Vision automatically on remote failure.
  • Opt-in, off by default. API key stored per-endpoint in Keychain. Sensitive and concealed pasteboard items are never sent to a remote service.
  • Localized in 7 languages.
