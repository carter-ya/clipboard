Clipboard 1.0.0

First public release.

New

  • Menu-bar-resident clipboard history panel, summoned by a global hotkey.
  • Four entry kinds — text / rich text / image / file — with thumbnail previews.
  • Filter by kind + full-text search (image OCR results are searchable too).
  • Sensitive-content detection: passwords, card numbers and the like stay in memory only — never written to disk, exported, or logged.
  • Per-source-app blocklist (bundle IDs).
  • AI summaries: Vision OCR + NaturalLanguage entity recognition; Foundation Models used when available on macOS 26+.
  • Localized UI: English / 简体中文 / 繁體中文 / 日本語 / 한국어 / Deutsch / Español.
  • Pinned items are exempt from the retention cap.
  • Import / export history as a zip archive.
  • Sparkle auto-update (EdDSA-signed, independent of Apple code signing).
  • Panel shortcuts: ⌘⇧[ / ⌘⇧] cycle kind tabs; ⌘1–⌘6 jump to a specific kind.
  • New Shortcuts pane in Preferences — all global and in-panel shortcuts in one place.

Improvements

  • Clipboard polling interval tightened to about 80ms — the miss window shrinks roughly 4×.

Distribution

  • Ships as an unsigned DMG; first run needs xattr -cr to clear the Gatekeeper quarantine.
  • Not on the Mac App Store; App Sandbox is not enabled.
