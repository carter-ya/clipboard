Clipboard 1.0.0

최초 공개 릴리스.

추가

  • 메뉴 막대 상주형 클립보드 기록 패널 — 전역 단축키로 호출합니다.
  • 텍스트 / 서식 있는 텍스트 / 이미지 / 파일 네 가지 항목 유형과 썸네일 미리보기.
  • 유형별 필터 + 전문 검색(이미지 OCR 결과도 검색 가능).
  • 민감 정보 감지: 비밀번호, 카드 번호 등은 메모리에만 남고 디스크·내보내기·로그에 노출되지 않습니다.
  • 소스 앱 기반 차단 목록(Bundle ID).
  • AI 요약: Vision OCR + NaturalLanguage 개체 인식. macOS 26+ 에서는 Foundation Models 도 활용.
  • 다국어 UI: English / 简体中文 / 繁體中文 / 日本語 / 한국어 / Deutsch / Español.
  • 고정한 항목은 보관 용량 제한에서 제외됩니다.
  • 기록을 zip 으로 가져오기 / 내보내기.
  • Sparkle 자동 업데이트(EdDSA 서명, Apple 코드 서명과 독립).
  • 패널 단축키: ⌘⇧[ / ⌘⇧] 로 유형 탭 순환, ⌘1–⌘6 으로 특정 유형 바로 이동.
  • 환경설정에 Shortcuts 탭 추가 — 전역 및 패널 단축키를 한곳에서 확인.

개선

  • 클립보드 폴링 주기를 약 80ms 로 단축하여 누락 구간을 약 4 배 축소.

배포

  • 서명 없는 DMG 로 배포합니다. 최초 실행 시 xattr -cr 로 Gatekeeper 격리 속성을 제거해야 합니다.
  • Mac App Store 에는 배포하지 않으며 App Sandbox 도 사용하지 않습니다.
