Clipboard 1.0.0

首次公開發佈。

新增

  • 選單列常駐的剪貼簿歷史面板，以全域快速鍵召喚。
  • 文字 / 富文字 / 圖片 / 檔案四種條目類型，附縮圖預覽。
  • 按 kind 過濾 + 全文搜尋（圖片 OCR 結果也可搜）。
  • 敏感內容識別：密碼、金融卡號等條目僅駐留記憶體，不落盤 / 匯出 / 日誌。
  • 按來源 App 封鎖（Bundle ID 黑名單）。
  • AI 摘要:Vision OCR + NaturalLanguage 實體識別；macOS 26+ 可選用 Foundation Models。
  • 多語系介面:English / 简体中文 / 繁體中文 / 日本語 / 한국어 / Deutsch / Español。
  • 置頂條目不受容量上限淘汰。
  • 歷史資料匯入 / 匯出為 zip。
  • Sparkle 自動更新（EdDSA 驗章，獨立於 Apple 簽章）。
  • 面板鍵盤快速鍵:⌘⇧[ / ⌘⇧] 循環切換 kind 標籤，⌘1–⌘6 直接選取 kind。
  • 偏好設定新增 Shortcuts 分頁:集中展示全域與面板內快速鍵。

優化

  • 剪貼簿輪詢間隔收緊至約 80ms，遺漏視窗縮小約 4×。

分發

  • 直接發佈未簽章 DMG；首次安裝需使用 xattr -cr 解除 Gatekeeper 隔離標記。
  • 不上 Mac App Store、不啟用 App Sandbox。
