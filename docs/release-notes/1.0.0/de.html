Clipboard 1.0.0

Erste öffentliche Veröffentlichung.

Neu

  • In der Menüleiste sitzender Verlauf der Zwischenablage, aufrufbar über einen globalen Tastenkürzel.
  • Vier Eintragstypen — Text / formatierter Text / Bild / Datei — mit Miniaturvorschau.
  • Filter nach Typ und Volltextsuche (auch OCR-Ergebnisse aus Bildern sind durchsuchbar).
  • Erkennung sensibler Inhalte: Passwörter, Kartennummern und Ähnliches bleiben ausschließlich im Arbeitsspeicher — niemals auf Festplatte, in Exporten oder Logs.
  • Sperrliste pro Quell-App (Bundle-IDs).
  • KI-Zusammenfassungen: Vision OCR + NaturalLanguage-Entitätserkennung; Foundation Models werden auf macOS 26+ genutzt, sofern verfügbar.
  • Mehrsprachige Oberfläche: English / 简体中文 / 繁體中文 / 日本語 / 한국어 / Deutsch / Español.
  • Angeheftete Einträge sind vom Speicherlimit ausgenommen.
  • Verlauf als ZIP-Archiv importieren / exportieren.
  • Sparkle-Auto-Update (EdDSA-signiert, unabhängig von der Apple-Codesignatur).
  • Panel-Shortcuts: ⌘⇧[ / ⌘⇧] wechseln zwischen Typ-Tabs; ⌘1–⌘6 springen direkt zu einem Typ.
  • Neuer Tab Shortcuts in den Einstellungen — alle globalen und Panel-Tastenkürzel an einer Stelle.

Verbesserungen

  • Abfrageintervall für die Zwischenablage auf ca. 80 ms verkürzt — das Zeitfenster für verpasste Einträge schrumpft etwa um den Faktor 4.

Distribution

  • Auslieferung als unsigniertes DMG; beim ersten Start muss das Gatekeeper-Quarantäne-Flag mit xattr -cr entfernt werden.
  • Nicht im Mac App Store; App Sandbox ist deaktiviert.
