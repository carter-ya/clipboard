Clipboard 1.0.0

初の一般公開リリース。

新機能

  • メニューバー常駐のクリップボード履歴パネル。グローバルホットキーで呼び出せます。
  • テキスト / リッチテキスト / 画像 / ファイルの 4 種類に対応、サムネイルプレビュー付き。
  • 種類でのフィルタと全文検索（画像の OCR 結果も検索可能）。
  • 機微情報の検出:パスワードやカード番号などはメモリ内にのみ保持し、ディスク保存・書き出し・ログには一切出力しません。
  • ソース App 単位のブロックリスト（Bundle ID）。
  • AI 要約:Vision OCR + NaturalLanguage による固有表現抽出。macOS 26+ では Foundation Models も活用します。
  • 多言語 UI:English / 简体中文 / 繁體中文 / 日本語 / 한국어 / Deutsch / Español。
  • ピン留めした項目は保持数の上限を超えても削除されません。
  • 履歴を zip でインポート / エクスポート。
  • Sparkle による自動アップデート（EdDSA 署名、Apple コード署名とは独立）。
  • パネルのキーボードショートカット:⌘⇧[ / ⌘⇧] でタブを循環、⌘1–⌘6 で特定の種類へ直接移動。
  • 環境設定に新しい Shortcuts タブ:グローバルとパネル内のショートカットを一覧できます。

改善

  • クリップボードのポーリング間隔を約 80ms に短縮、取りこぼし窓を約 4 分の 1 に縮小。

配布

  • 未署名 DMG として配布。初回起動時は Gatekeeper の隔離属性を xattr -cr で解除する必要があります。
  • Mac App Store では配布しません。App Sandbox も有効にしません。
